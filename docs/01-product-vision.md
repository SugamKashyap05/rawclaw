# Product Vision

## Mission

RawClaw is a secure, local-first AI agent platform that helps users research, automate, execute, and review work on their own machine with transparency and control.

## Core promise

Users should be able to:
- talk to the system naturally
- reach the system from the surfaces they already use
- have the system use real tools instead of guessing
- run repeatable tasks with saved outputs
- inspect what happened, what tools were used, and what sources were trusted

## User experience goals

### Trusted outputs
- Current claims should be verified, sourced, and dated
- If the system cannot verify something, it must say so clearly

### Strong operations view
- The UI should feel like a command center
- Chat, tasks, MCP, memory, models, and health should all be visible

### Local control
- Local models, local data, and local runtimes should be first-class
- Cloud integrations should be optional, not required
- A single local control plane should remain the source of truth for routing, sessions, trust, and runtime operations

### Productive automation
- Users should be able to run one-off tasks and recurring tasks
- Runs should produce polished outputs, not debug dumps

## Product pillars

1. Chat
2. Gateway and channels
3. Tools and MCP
4. Tasks and artifacts
5. Memory and RAG
6. Specialists and delegation
7. Local operations and visibility

## Non-goals

- A generic chatbot with no provenance
- Opaque background automation with no auditability
- A system that depends entirely on cloud-only services
